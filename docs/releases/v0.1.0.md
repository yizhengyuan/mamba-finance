# Mamba Finance v0.1.0

首个可验收的 MVP 版本，覆盖借贷业务主链路，并完成自动化测试与手工验收。

## 版本亮点

- 完成订单全流程：创建订单、查看详情、状态跟踪
- 完成回款主流程：还款计划生成、核销回款、状态联动
- 提供经营看板：资产趋势、构成、到期与逾期结构
- 支持账户管理：账户创建、编辑、启停管理
- 支持还款日历：按月查看计划、关键词筛选、抽屉核销
- 支持附件墙：订单附件查看与管理

## 产品与体验升级

- 前端主界面完成中文化
- 新增 UI 设计实验室 `/ui-lab`
- 抽象通用 UI 组件（按钮、表单、状态标签、卡片）并迁移到关键页面
- README 改写为客户视角产品介绍

## 工程能力与文档

- 新增一键本地启动脚本：`npm run dev:up`
- 新增临时分享脚本：`npm run dev:share`（演示用途）
- 新增测试与验收手册
- 新增技术手册，拆分开发/部署信息

## 验收与质量状态

- 自动化测试通过：`unit` + `integration` + `e2e`
- 手工验收通过：账户 -> 订单 -> 核销 -> 日历 -> 看板
- 版本标签：`v0.1.0`

## 已知说明

- 当前为 MVP 阶段，适合试运行与业务演示
- “AI 全自动财务中台”能力已归档到路线图，后续版本再推进
