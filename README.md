# Mamba Finance

Mamba Finance 是一套面向借贷业务的资产与债权管理产品，帮助团队把“订单创建、还款计划、回款核销、经营看板”放在同一套系统里管理。

## 产品定位

面向小微借贷/典当/分期等场景，解决三类核心问题：

- 账务分散：订单、账户、回款记录分散在多个表格或聊天记录中
- 回款跟踪难：到期、逾期、实收状态无法实时统一
- 经营不可视：难以快速看到资产结构、逾期风险与资金变化

## 核心价值

- 一处录入，全链路追踪：从订单创建到回款核销，全程可追溯
- 状态自动同步：回款后自动联动订单、计划、账户余额
- 看板即决策：用可视化指标快速识别风险和经营趋势

## 主要功能

- 订单管理：创建订单、查看详情、跟踪状态
- 还款日历：按月查看到期计划，支持关键词筛选
- 回款核销：直接选择账户完成核销，自动更新状态
- 账户管理：统一管理收款账户与启停状态
- 经营看板：总资产、到期与逾期结构一屏呈现
- 附件墙：集中查看与管理订单相关附件

## 典型使用流程

1. 新建收款账户  
2. 新建借款订单并生成还款计划  
3. 在订单详情或还款日历中执行核销  
4. 在经营看板查看资产与风险变化  

## 当前阶段

当前版本已覆盖 MVP 主链路，可用于内部试运行与业务流程演示。  
“AI 全自动财务中台”相关能力已沉淀为后续路线图，暂不在本阶段实现。
